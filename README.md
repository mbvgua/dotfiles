# dotfiles

> If you only knew the power of the dark side. 
> ~ Darth Vader.

This repo houses dotfiles for my main developer environment:
    1.vim -> text-editor
    2.tmux -> terminal multiplexer
    3.bash -> terminal
    4.git -> version controll

Alot of ideas and inspirations were gotten from the internet generally so I'll be sure to give the links to those. Also checkout the [r/vim](https://www.reddit.com/r/vim/) and [r/neovim](https://www.reddit.com/r/neovim/) on reddit. They have a lot of nice hacks and inspirational rices.

Not alot of customization was done to git, bash or tmux in terms of general appearance; Vim is where the bulk of the styling happened.
